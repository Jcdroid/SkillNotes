# 工具-Visual Studio Code

### 调整编辑窗口为绿豆沙颜色
1. 安装`Brackets Light Pro`theme插件；
2. 在`C:\Users\[你的用户名]\.vscode\extensions`下找到`eryouhao.brackets-light-pro\theme\Brackets Light Pro-color-theme.json`;
3. 把`colors`下的`editor.background`修改为`#c7edcc`；
4. 重启VSCode即可

### 常用快捷键
* 代码格式化：`shift + alt + f`
* 复制行：`shift + alt + up/down`
* 删除行：`ctrl + shift + k`
* 全局查找文件：`ctrl + shift + f`
* 显示git：`ctrl + shift + g`
* 新建一个窗口 : `ctrl + shift + n`
* 控制台终端显示与隐藏：`ctrl + ~`
* 查找文件/安装vs code 插件地址：`ctrl + p`
* 快速回到顶部： `ctrl + home`
* 快速回到底部：`ctrl + end`
* 移动行：`alt + up/down`
* 显示/隐藏左侧目录栏：`ctrl + b`
* 字体放大/缩小: `ctrl + ( + 或 - )`
* 打开最近打开的文件：`ctrl + r`
* 查找：`ctrl + f`
* 替换：`ctrl + h`
